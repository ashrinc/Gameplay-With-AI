# NOTICE: FRONTEND DEPENDENCIES
# ---------------------------------------------------------
# The dashboard is built with Node.js and React (JavaScript), 
# which means it DOES NOT use Python's "pip" or requirements.txt natively.
#
# The actual "requirements file" for this dashboard is: package.json
#
# HOW TO INSTALL:
# 1. Open terminal and go to the dashboard folder: cd dashboard
# 2. Run the Node Package Manager install command: npm install
# 3. Start the dashboard server: npm start
#
